üéØ Where to Find It

  1. Go to Strapi Admin Panel
  2. Click "Projects" in the sidebar
  3. Select your project (e.g., "bouar 673")
  4. Find the "Property Type Info" field
  5. Replace the JSON content with the structure below

üìù JSON Structure

{ "simplex": { "range": "60-170m", "status": "sold_out", "image": "https://images.unsplash.com/photo-1560448204-e02f11c3d0e2?w=400&h=300&fit=crop" }, "duplex": { "range": "110-270m", "status": "sold_out", "image": "https://images.unsplash.com/photo-1564013799919-ab600027ffc6?w=400&h=300&fit=crop" } }

üîß Fields Explained

  • range: Size range (e.g., "60-170m", "110-270m")
  • status: Must be exactly "available" or "sold_out"
  • image: Full URL to property type image (optional but recommended)

üìã Important Notes

‚úÖ Order Matters: Property types display in the SAME ORDER as they appear in this JSON. First property type = first displayed on website.
‚ö†Ô∏è Status Values: Use exactly "available" or "sold_out" (lowercase with underscore). "Sold_out" or "Available" won't work.

üí° Tips

  • Use only property types that exist in your Property Types collection
  • Images are optional but make property types more attractive
  • Status "available" shows WhatsApp button, "sold_out" hides it
  • Use full URLs for images (starting with http:// or https://)
